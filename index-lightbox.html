Lightbox2 example

Uses the jquery lightbox2 plugin from lokeshdhakar.com. This plugin doesn't load lightbox2, it uses it if you already have lightbox2 on your site. (at the time of writing lightbox2 didn't work with jquery 1.9)

$(document).ready(function(){
    $("ul.youtube-videogallery").youtubeVideoGallery( {plugin:'lightbox'} );
});

NB: If you want to add lightbox functionality and don't already have anything installed, this plugin works very well with colorbox form www.jacklmoore.com.
